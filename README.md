# GestorGastos

App Android (Kotlin + Jetpack Compose + Room) para registrar gastos personales:
- Registrar gasto: monto, moneda, concepto, método de pago, comercio, dirección, descripción
- Adjuntar recibo (Picker / Cámara)
- Exportar CSV a Drive
- Bloqueo por PIN

## Requisitos
- Android Studio (última versión estable)
- Android SDK instalado por Android Studio

## Cómo ejecutar
1. Abrir el proyecto en Android Studio
2. Sync Gradle
3. Run en emulador o dispositivo Android
